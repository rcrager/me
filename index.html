Projects

Disk Galaxy Simulation

This was for a Computational Physics course I took at UMass. We needed to make a Python program that simulated a disk galaxy remaining stable under its own gravity within an NFW dark matter halo. We then used an energy-conserving algorithm (the leapfrog method) to evolve the simulation in time and showcase a minor merger event - the code is available on my Github

Research

Are certain types of galaxies more prone to hosting an Active galactic Nucleus (AGN)?

There are a wide variety of galaxies thriving all around us. We can learn more about these by quantifying the physical properties of them through a number of ways (i.e., morphological analysis, spectroscopy, etc.) This is one of the motivating questions behind my current research project with Professor Alex Pope.

How can we measure these?

We quantify these structural features in the galaxies by calculating their nonparametric measurements using a popular library for this purpose: Statmorph. Using Statmorph we can see statistical trends that represent visual and quantitative structures within the galaxy. This is particularly useful in our case since we're looking for any correlations between structural features and AGN fraction.

In addition to the morphological analysis, we search for visual differences in structure. We do this by obtaining main classifications (Disk, Spheroid, or Irregular) of these galaxies and note specific structural features (clumps, mergers, spiral arms) for each to search for correlations later.
